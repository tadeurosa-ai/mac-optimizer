Instalação passo a passo

Nunca usou o Terminal? Sem problema. Siga os passos abaixo.

Instalar o Node.js

Abra o Terminal (Cmd + Espaço → Terminal → Enter) e cole:

brew install node
Se aparecer "command not found: brew", primeiro instale o Homebrew:
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Depois repita o brew install node.

Instalar o Claude Code

npm install -g @anthropic-ai/claude-code

Aguarde 1–2 minutos até terminar.

Fazer login

claude

Na primeira vez vai pedir login com sua conta Claude (mesma do site). Siga as instruções.

Instalar o Mac Optimizer

mkdir -p ~/.claude/commands && curl -fsSL https://raw.githubusercontent.com/tadeurosa-ai/mac-optimizer/main/optimizer.md -o ~/.claude/commands/optimizer.md && echo "✓ /optimizer instalado"

Só precisa fazer isso uma vez. O comando fica salvo para sempre.

Usar

Abra o Claude Code (claude no Terminal) e digite:

/optimizer

Claude conduz tudo. Você só responde as perguntas.

Requisitos

macOS 12 ou superior Claude Code Plano Pro ou Max

Dúvidas

Preciso pagar algo além do Claude Pro/Max? Não. O plugin é gratuito.
É seguro? Sim. Claude só deleta com sua confirmação explícita. Itens duvidosos são perguntados antes.
Funciona toda vez? Sim. Após instalar, basta digitar /optimizer em qualquer sessão do Claude Code.
Como desinstalar? 
rm ~/.claude/commands/optimizer.md
